# Superstore-Sales-Profit-Analysis
This project focuses on data preprocessing of a raw dataset using NumPy and Pandas, followed by exploratory data analysis using Matplotlib. Finally, interactive dashboards were built in Power BI to visualize key business insights and support decision-making.
This project is analyzes sales, profit, discount, and shipping performance of a retail dataset.
The goal is to identify key business insights such as profitable segments, loss-making products, and the impact of discounts on profit.
